Part I - Babbling Brook

This implements the lab’s SuperCollider idea using brown noise, low-pass filtering, and a high-pass filter whose cutoff is modulated by another low-passed brown-noise signal.

Brook Signal Flow

BrownNoise → Lowpass → Highpass → Gain → Destination ↑ BrownNoise → Lowpass → Gain(depth) + Constant(offset) → Highpass.frequency

Part II - Alarm Generator

This alarm uses separate oscillators for fixed pitches and switches between them with smoothed gain changes so the tones alternate cleanly without clicks.

Current sequence: 800, 600

Alarm Signal Flow

Oscillator(voice 1) → Gain ┐ Oscillator(voice 2) → Gain ├→ Sum → Highpass → Master Gain → Destination Oscillator(voice 3) → Gain ┘ Sequencer / timer → smooth gain changes → selects which tone is heard
